    Welcome to Ensidia

    Ensidia is a small Unix-like clone. We'll see how it goes.

    To play with it, make sure you have qemu installed, and that you're on a Unix system:

    git clone https://github.com/NlightNFotis/Ensidia.git
make qemu

    and have fun!

    Folder structure

    • bin/ contains the common Unix userspace utilities
    • doc/ may (or may not :D ) contain some documentation
    • include/ is where all the header files are
    • lib/ is where the libc is located
    • test/ includes some userspace testing software
    • utils/ includes some utility software that's used to document or build the software
    • xng/ is where the Ensidia kernel files are located.
